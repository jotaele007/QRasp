# Android RASP SDK

Lightweight Runtime Application Self-Protection SDK for Android.
Developed by JL

## Features
- Debugger detection
- Root detection
- Emulator detection
- Hook / Frida detection
- App tampering detection

## Usage
```kotlin
Rasp.check(this, RaspConfig(killAppOnThreat = true))
```
